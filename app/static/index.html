          Actor Types (Actuator hardware catalog)

          Global catalog shared across every project — e.g. a "Schaltaktor" from MDT with 20 channels. The "Type" must match a Point Type's channel type (Setup tab) to be assignable to it in a project's Abgangsliste.

            Categories (Main Groups)

            Your standard function categories. Order = Main Group number. Usually set once and left alone.

              Point Types

              Reusable building blocks (a light, a blind, a socket...) with their datapoints and reserved block size. "Channel type" links a point to matching Actor Types below, so it can be wired to a physical actuator channel in a project's Abgangsliste.

                Central / General Function Templates

                Auto-generated every project. "building" scope = one block for the whole project. "floor" scope = one block per floor (e.g. "Zentral EG", "Zentral OG"). "room_multi" scope = one block PER ROOM, only for rooms that have several points in this category (e.g. a room with 2+ blinds gets its own combined "Zentral" control). Allgemein category templates each become their own Middle Group.
